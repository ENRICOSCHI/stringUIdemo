# Modello fisico chitarra

Questo modello fisica è stato creato in JUCE e può essere suonato sia con il mouse che con la tastiera MIDI.

# Descrizione

## Generazione del rumore

Questo modello fisico usa l'algoritmo ideato da Karplus-Strong per simulare il suono della chitarra.

## Effetti

Il modello presenta tre effetti attivabili e disattivabili dalla UI: **Distorsione, Delay e Riverbero**.
